# Коллекция сайтов по изучению русского языка как иностранного

Эта коллекция содержит архивы веб-сайтов, связанных с изучением русского языка как иностранного. Архивы были собраны с использованием **wpull**, а метаданные и анализ коллекции получены с помощью **metawarc** и сервиса **ArchiveReady**.
